# Cifrado y Análisis Combinatorio

Proyecto desarrollado para la Materia Matemáticas 2.
Pequeña aplicación en un solo archivo que demuestra un cifrado por sustitución usando una permutación completa del alfabeto (monoalfabético). Ideal para mostrar rápidamente en tu perfil.

Cómo usar
1. Descarga o clona el repositorio.
2. Abre el archivo `Cifrado-y-Analisis-Combinatorio.html` en tu navegador.
3. Genera una llave aleatoria o importa una llave de 26 letras.
4. Escribe el texto a encriptar o pega el texto cifrado, y alterna "Encriptar"/"Desencriptar".

Características
- Llave como permutación completa de las 26 letras A–Z.
- Encriptado y desencriptado por sustitución monoalfabética.
- Mantiene signos, números y espacios sin cambios.
- Copiar/descargar/importar llave y copiar resultado.
